Jul 20, 2026

Template showing how to embed images and inline SVG diagrams.

For an image, use a <figure> with an <img> and a <figcaption>. Drop the file at /img/blog/<post-slug>/<filename>.

For a diagram, use inline SVG. It scales perfectly, prints cleanly, and doesn't require an external file. The pattern below uses three boxes and arrows — copy and edit the <rect> and <text> elements.

Both elements work in print mode, scale on mobile, and don't require any JavaScript.
