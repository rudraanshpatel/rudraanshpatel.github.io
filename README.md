# rudraanshpatel.github.io
 
Personal portfolio website of **Rudraansh Patel** — CSE student, software engineer, and competitive programmer.
 
🌐 **Live at [rudraanshpatel.in](https://rudraanshpatel.in)**
 
---
 
## Overview
 
A minimal, clean portfolio built with vanilla HTML and CSS — no frameworks, no dependencies, just fast and lightweight. Features smooth scroll-reveal animations, a sticky nav, and full mobile responsiveness.
 
## Sections
 
- **Hero** — Introduction and quick links
- **Projects** — Showcase of personal and academic projects
- **Skills & Stack** — Languages, frameworks, and tools
- **Competitive Programming** — Profiles and ratings on Codeforces, LeetCode, and CodeChef
- **Contact** — Email, GitHub, and LinkedIn
## Tech Stack
 
- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript (Intersection Observer for scroll animations)
- Google Fonts — DM Serif Display & DM Sans
- Hosted on **GitHub Pages** with a custom domain via GoDaddy
## Local Development
 
No build step needed. Just clone and open:
 
```bash
git clone https://github.com/rudraanshpatel/rudraanshpatel.github.io.git
cd rudraanshpatel.github.io
# Open index.html in your browser
```
 
Or use VS Code's Live Server extension for hot reload.
